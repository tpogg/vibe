# Agent Knowledge System

An interconnected knowledge graph exploring how agents think through external structures. Just as Zettelkasten, evergreen notes, and memory palaces gave humans external scaffolding for thought, agent-operated knowledge systems give agents external structures to think with.

## How to Navigate

- **Wiki links** (`[[note-name]]`) connect ideas across the graph
- **MOCs** (Maps of Content) serve as topical entry points
- **Synthesis notes** develop arguments about how pieces fit together
- **Claims** capture cross-domain insights that bridge topic areas
- **Explorations** track open questions and research directions
