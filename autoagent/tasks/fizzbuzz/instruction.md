# FizzBuzz

Write a Python script that generates FizzBuzz output for numbers 1 through 100 and saves it to `/app/output/fizzbuzz.txt`.

Rules:
- For multiples of 3, print "Fizz"
- For multiples of 5, print "Buzz"
- For multiples of both 3 and 5, print "FizzBuzz"
- For all other numbers, print the number itself

Each entry should be on its own line. The file must contain exactly 100 lines.
