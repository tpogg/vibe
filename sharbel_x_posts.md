## Post 11: How I Built a Trading Bot With Zero Code

**Date:** ~January 2026
**URL:** [x.com/sharbel/status/2020907505929556217](https://x.com/sharbel/status/2020907505929556217)

> how i built a trading bot with zero code:
>
> 345 wins. 26 losses. 93% win rate.
>
> no python knowledge. no stackoverflow. just me talking to an AI agent and iterating.
>
> here's the playbook:
>
> 1. start with the dumbest version possible.
>    my first prompt was literally "build me a bot that bets on polymarket 15-minute markets"
>    -- it lost $11 on the first trade.
>
> 2. every time the bot lost money, i'd paste the trade log into Claude and ask
>    "why did this lose? fix it."
>    each loss = one prompt = one fix.
>
> 3. the "brain + muscles trick":
>    - expensive model for complex decisions
>    - cheap model for monitoring
>    - free rule-based logic for actual trades
>    this reduced AI costs from $300/month to $30/month.
>
> 4. set up a cron job that runs after every loss, reads the log, analyzes what went wrong,
>    and suggests fixes -- the AI debugging itself.
